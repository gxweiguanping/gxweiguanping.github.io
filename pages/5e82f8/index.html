CentOS7离线安装 Docker

# CentOS7离线安装 Docker

在 CentOS 7 系统中离线安装 Docker，你需要先从有网络连接的环境下载 Docker。以下是具体的步骤：

# 步骤1：从Docker 的静态二进制存档页面下载 Docker

你可以访问 https://download.docker.com/linux/static/stable/x86_64/ (opens new window)下载

# 步骤2：通过 FTP工具上传 docker-18.06.1-ce.tgz 到服务器上的Home目录

# 步骤3：解压安装包到Home目录

tar zxf docker-18.06.1-ce.tgz
1

# 步骤4：将docker 相关命令拷贝到 /usr/bin，方便直接运行命令

sudo cp docker/* /usr/bin/
1

# 步骤5：启动Docker守护程序

sudo dockerd &
1

# 步骤6： 验证 Docker 安装

最后，你可以运行 docker version 命令来验证 Docker 是否成功安装：

docker version
1

如果这个命令输出了 Docker 的版本信息，那么说明 Docker 已经成功安装。

# 步骤7：配置加速镜像

目前主流 Linux 发行版均已使用 systemd (opens new window) (opens new window)进行服务管理，这里介绍如何在使用 systemd 的 Linux 发行版中配置镜

像加速器。

1、请首先执行以下命令，查看是否在 docker.service 文件中配置过镜像地址。

systemctl cat docker | grep '\-\-registry\-mirror'
1

2、如果该命令有输出，那么请执行 $ systemctl cat docker 查看 ExecStart= 出现的位置，修改对应的文件内容去掉 --registry-mirror 参数及其值，并按接下来的步骤进行配置。

3、如果以上命令没有任何输出，那么就可以在 /etc/docker/daemon.json 中写入如下内容（如果文件不存在请新建该文件）：

{
  "registry-mirrors": [
    "http://hub-mirror.c.163.com",
    "https://registry.docker-cn.com",
    "https://y4xpdpoy.mirror.aliyuncs.com"
  ]
}

# 步骤8：将docker注册成系统服务（记得kill docker服务后，再执行这一步哦）

sudo vi /usr/lib/systemd/system/docker.service
1
[Unit]
Description=Docker Application Container Engine
Documentation=https://docs.docker.com
After=network-online.target firewalld.service
Wants=network-online.target
 
[Service]
Type=notify
ExecStart=/usr/bin/dockerd
ExecReload=/bin/kill -s HUP $MAINPID
LimitNOFILE=infinity
LimitNPROC=infinity
TimeoutStartSec=0
Delegate=yes
KillMode=process
Restart=on-failure
StartLimitBurst=3
StartLimitInterval=60s
 
[Install]
WantedBy=multi-user.target

# 步骤9： 停止 docker 服务

systemctl stop docker
1

# 步骤10：开机自启docker 服务

systemctl enable docker
1

# 步骤11：修改 docker 的配置，开放远程部署端口

vi /lib/systemd/system/docker.service
1

其中 ExecStart= 后添加配置：

-H tcp://0.0.0.0:2375 -H unix:///var/run/docker.sock
1

# 步骤12：启动 Docker 服务

你需要启动 Docker 服务。你可以使用以下命令来完成这个步骤：

systemctl start docker
1

# CentOS7离线安装 docker-compose

# 下载安装包

docker-compose软件包下载地址 (opens new window)

下载位置可选，我这里是/data/software/

# 添加文件权限

chmod +x /data/software/docker-compose
1

# 创建软链接

ln -s /data/software/docker-compose /usr/bin/docker-compose
1

# 验证安装是否成功

docker-compose --version
1
