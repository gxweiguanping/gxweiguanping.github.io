kafka内外网分流

# kafka内外网分流

# 描述

需要部署一台能外网访问的kafka进行联调，公司的外网环境是通过net端口映射进来的，把内网的服务器的部分端口映射到一台外网服务器的端口上。

举个例子就是如果把服务器上的tomcat端口8080映射到外网的9090，那么外网就应该使用外网ip:9090访问，通过防火墙映射之后实际访问的就是内网ip:8080

本次使用的是[wurstmeister/kafka](GitHub - wurstmeister/kafka-docker：Apache Kafka 的 Dockerfile (opens new window))镜像。

# docker-compose方式

配置文件：docker-compose.yml

version: '3'
services:
  zookeeper:
    image: wurstmeister/zookeeper
    container_name: zookeeper
    ports:
      - 2181:2181
    restart: always
    privileged: true
  kafka1:
    image: wurstmeister/kafka
    container_name: kafka_master
    ports:
      - "29092:29092"
      - "9092:9092"
    environment:
      KAFKA_BROKER_ID: 5  
      KAFKA_NUM_PARTITIONS: 2
      KAFKA_DEFAULT_REPLICATION_FACTOR: 2
      KAFKA_ZOOKEEPER_CONNECT: 192.168.1.62:2181
      KAFKA_LISTENERS: INSIDE://:9092,OUTSIDE://:29092
      KAFKA_ADVERTISED_LISTENERS: INSIDE://192.168.1.62:9092,OUTSIDE://{外网ip}:29092
      KAFKA_LISTENER_SECURITY_PROTOCOL_MAP: INSIDE:PLAINTEXT,OUTSIDE:PLAINTEXT
      KAFKA_INTER_BROKER_LISTENER_NAME: INSIDE
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
    restart: always
    privileged: true

# 参数解释

参数 示例值 说明
KAFKA_LISTENERS INSIDE://:9092,OUTSIDE://:29092 定义了 Kafka 监听器，包括两个监听器，一个在 INSIDE（内部）地址使用端口 9092，另一个在 OUTSIDE（外部）地址使用端口 29092。
KAFKA_ADVERTISED_LISTENERS INSIDE://192.168.1.62:9092,OUTSIDE://{外网ip}:29092 Kafka 服务器在元数据中公布给客户端的监听地址
KAFKA_LISTENER_SECURITY_PROTOCOL_MAP INSIDE:PLAINTEXT,OUTSIDE:PLAINTEXT 定义了监听器使用的安全协议。在这里，INSIDE 和 OUTSIDE 的监听器都使用的是明文（PLAINTEXT）协议
KAFKA_INTER_BROKER_LISTENER_NAME INSIDE kafka集群时内部访问使用的默认方式

# docker命令方式

docker run -d \
  --name kafka_master \
  -p 29092:29092 \
  -p 9092:9092 \
  -e KAFKA_BROKER_ID=5 \
  -e KAFKA_NUM_PARTITIONS=2 \
  -e KAFKA_DEFAULT_REPLICATION_FACTOR=2 \
  -e KAFKA_ZOOKEEPER_CONNECT=192.168.1.62:2181 \
  -e KAFKA_LISTENERS=INSIDE://:9092,OUTSIDE://:29092 \
  -e KAFKA_ADVERTISED_LISTENERS=INSIDE://192.168.1.62:9092,OUTSIDE://{外网ip}:29092 \
  -e KAFKA_LISTENER_SECURITY_PROTOCOL_MAP=INSIDE:PLAINTEXT,OUTSIDE:PLAINTEXT \
  -e KAFKA_INTER_BROKER_LISTENER_NAME=INSIDE \
  -v /var/run/docker.sock:/var/run/docker.sock \
  --restart always \
  wurstmeister/kafka
