# IDEA常用快捷键

快捷键组合 实现效果
psvm + Tab键 / main + Tab键 public static void main(String[] args)
sout + Tab键 System.out.println()
Ctrl + X 删除当前行
Ctrl +D 复制当前行
Alt+Insert(或右键Generate) 生成代码(如get,set方法,构造函数等)
Alt+Ctrl+T 生成try catch （或者 Alt+enter选择）
Ctr+shift+U 实现大小写之间的转化
ALT+回车 导入包并自动修正
CTRL+ALT+L 格式化代码
fori 生成for (int i = 0; i < ; i++) {}
Alt + <–左右–>键 实现窗口左右更换（多窗口）
Ctrl + 鼠标点击 快速找到成员变量的出处
Shift+F6 重构/重命名 (包、类、方法、变量、甚至注释等)
CTRL+Q 查看当前方法的声明
Ctrl+O 重写方法
Ctrl+I 实现方法
Ctrl+P 显示方法参数信息
ALT+/ 代码提示
Ctrl+Shift+Insert 查看历史复制记录，idea可以保留历史复制的 100 条记录
Ctrl + Y 删除整行
Ctrl + W 快速选中

# IDEA常用插件

插件名称 作用
CodeGlance 拖动浏览代码更加方便，还有放大镜功能。
Rainbow Brackets 它可以实现配对括号相同颜色，并且实现选中区域代码高亮的功能。对增强写代码的有趣性和排错等都有一些帮助。
Translation 很不错的翻译插件，支持谷歌、有道、百度三种翻译。
Alibaba Java Code Guidelines 阿里巴巴出品的代码规范插件，用于帮助程序员规范自己的代码，检测出潜在的问题，改善代码质量。
Maven Helper 查看maven的依赖树。
MybatisX MybatisX 是一款基于 IDEA 的快速开发插件，为效率而生。最强拍档：搭配Mybatis-Plus开发。具备XML 跳转功能。
git commit template git提交规范插件。
Mybatis Log Free 用于记录MyBatis执行的SQL语句。
PlantUML Integration PlantUml画图利器。
Smart Input 注释的时候能自动切换中文，编写代码的时候自动切换英文的插件。
Grep Console 给控制台日志不同的颜色，日志管理神器。

# IDEA中如何查看现有分支是基于哪个分支创建的?

通常情况都是在master分支上创建新的分支,开发完成后合并到master上.

有的时候刚创建完新的分支,却不记得基于哪个分支创建的了.IDEA上比较容易查基于哪个分支.

1.直接打开下方的"Terminal"栏,快捷键是ALT+F12

2.输入 git reflog --date=local

3.Ctrl + F 查询现有版本

4.找到 checkout: moving from feature/基于的分支 to feature/新创建的分支 类似这样的

// 这里就是代表 "内容管理"分支是基于"成员管理"创建的新分支,而不是基于master分支
HEAD@{Thu Aug 18 17:23:01 2021}: checkout: moving from feature/V1.1.1成员管理 to feature/内容管理
